Formatting Example (+ Span & Div)

We will be using different formatting types to display some 'C++ Code' in this webpage. We will use the <code> and <pre> tag to display monospaced typing with unformatted whitespaces while using the <span> tag to perform syntax highlighting. To create a Text Window effect, we will use containers with the <div> tag and a 'little' bit of CSS styling.

Let us start with a simple Hello World program:

hello-world.cpp 

                        
// The Greatest Program of All Time: Hello World                
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    return 0;
}


We could also make a Simple Counter program using a loop:

loop-counter.cpp 


// Simple Counter to a Limit
#include <iostream>
using namespace std;

int main() {
    
    int limit = 10;
    cout << "From 1 to " << limit << ": ";
    for (int i; i < limit; i++) {
        cout << i + 1 << " ";
    } 
    return 0;
}
